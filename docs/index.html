常见问题

和 ChatGPT 有什么区别？
ChatGPT 是聊天网页。Claude Code 是能操控你电脑的 AI 助手——它能读写你的文件、执行命令、操控浏览器、甚至通过微信跟你对话。
要花钱吗？模型贵不贵？
源码免费、安装免费。DeepSeek V4 按量计费，输入 1 元/百万字，输出 2 元/百万字。个人日常用一个月几块钱。新用户注册送额度。
微信接入需要什么？
需要 iPhone + 微信最新版（ClawBot 插件）。Android 暂不支持腾讯官方 iLink 协议。
安全吗？会不会偷我代码？
全部源码开源在 GitHub，每一行都能看到。API Key 只存在你本地 ~/.claude/settings.json，代码永远不上传任何第三方服务器。

远程安装时当场测试，确认能用才走。装不了全额退款。尊享版 30 天内任何问题免费重装。
能用其他模型吗？不是 DeepSeek 可以吗？
可以。Claude Code 支持任何 Anthropic Messages API 兼容的端点。想用 OpenAI、Groq、Ollama 本地模型都行，改一下配置文件即可。
